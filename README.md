Como crear una API en firebase

## paso 1
  
  instalar npm y node para window o linux

## paso 2

  instalar firebase tools - ejemplo para linux
    
    sudo npm intall -g firebase-tools

## paso 3

  iniciar sesion en firebase. siga los pasos 
  
    firebase login
